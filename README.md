# Computational-Math-Solvers

This repository contains implementations of advanced mathematical algorithms using SageMath and symbolic computation. The goal is to provide optimized solvers for problems in Linear Algebra, Number Theory, and Cryptographic mathematics.

## Features
- **Linear Algebra:** Automated matrix diagonalization and Jordan Normal Form computation.
- **Number Theory:** Implementation of the Chinese Remainder Theorem and Primality Testing.
- **Symbolic Calculus:** Automated Taylor Series expansions and ODE solving.

## Environment Setup
These scripts are designed to run in a SageMath environment. To run these in Google Colab, use the installation instructions provided within the notebooks.
